# 最长公共子序列

## 题目描述

给定序列 *A*=(*a*1,*a*2,…,*an*) 和 *B*=(*b*1,*b*2,…,*bm*) ，求它们的最长公共子序列

子序列的定义参考题目  **最长上升子序列**

## 输入格式

第一行两个整数 *n*,*m* 代表序列 *A*,*B* 的长度

第二行是 *n* 个由空格隔开的整数，代表 *a*1,*a*2,…,*an*

第三行是 *m* 个由空格隔开的整数，代表 *b*1,*b*2,…,*bm*

## 输出格式

输出一个整数，代表最长公共子序列的长度

## 测试样例

### 样例输入

```
5 5
3 2 1 4 5
1 2 3 4 5
```

### 样例输出

```
3
```

## 数据规模

对于 100% 的数据，1≤*n*,*m*≤5000,1≤*ai*,*bi*≤10^4
