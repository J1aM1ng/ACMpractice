# 矩阵选数

## 题目描述

给定一个 3 行，*n* 列的矩阵，我们要在矩阵的每一列选一个数。对于第 *i*(1≤*i*≤*n*) 列，我们令 *di* 为第 *i* 列选择的数。那么，∑1~*n*−1∣*di*−*d(i*+1)∣ 最小是多少

## 输入格式

第一行一个整数 *n* ，描述见上文

后面三行每行 *n* 个整数，为矩阵的各个元素

## 输出格式

一个整数，题目要求的最小值

## 测试样例

### 样例输入

```
5
5 10 5 4 4
1 7 8 4 0
3 4 9 0 3
```

### 样例输出

```
3
```

## 数据规模

对于 100% 的数据，1≤*n*≤106 ，矩阵中数据绝对值不超过 10^6
