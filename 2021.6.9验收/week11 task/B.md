# 拿数问题

## 题目描述

给定 *n* 个数，我们要从中选出多若干个数，其中任意**大小不同的两数**差的绝对值不能为 1，那么选出的数之和最大是多少。

## 输入格式

第一行一个整数 *n* 。

第二行 *n* 个整数，*a*1,*a*2,...,*an* ，表示这 *n* 个数

## 输出格式

一个整数，选出的数的和的最大值

## 测试样例

### 样例输入

```
5
1 1 3 2 3
```

### 样例输出

```
8
```

## 数据规模

对于100% 的数据，1≤*n*≤10^6,1≤*ai*≤10^6
